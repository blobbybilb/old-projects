Enripen API Docs

POST /api/find

Details

Takes JSON POST params to find saved URLs. Returns JSON in the format: { <path>: [{ linkFrom: string, linkTo: string, createdAt: Date, submittersCount: number }] }.

POST Parameters

URL string

POST /api/save

Details

Takes JSON POST params to save a new link. Returns 'Saved!'.

POST Parameters

oldURL string

newURL string
